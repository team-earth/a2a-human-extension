# LaTeX Source for HLDM'25 Paper

This folder contains the LaTeX source code for the conference paper:

**"Integrating Civic Initiatives into AI Agent Ecosystems: A Human-Aligned Extension of the A2A Protocol"**

Presented at HLDM'25 Workshop (ECML PKDD 2025), Porto, Portugal, September 15, 2025.

## Files

- `paper.tex` - Main LaTeX source file
- `llncs.cls` - Springer LNCS document class
- `splncs04.bst` - Bibliography style file
- `readme.txt` - Original author kit readme

## Compiled PDF

The compiled PDF is available at: [HLDM2025-A2A-Human-Extension.pdf](../HLDM2025-A2A-Human-Extension.pdf)
