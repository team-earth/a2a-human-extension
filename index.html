ProgramCard Schema

Complete schema for describing human-led civic programs, organizations, and initiatives with G-O-S-R integration, operating character, and participation templates.

Operating Character Schema

Schema for encoding organizational style, motivation, and interaction patterns using Spiral Dynamics, Hofstede dimensions, and Insights Discovery frameworks.

https://team-earth.github.io/a2a-human-extension/schemas/operating-character/v1/schema.json View Schema →

Participation Template Schema

Schema for structured guidance on how to join, replicate, support, or engage with programs. Enables relational and adaptive participation beyond transactional APIs.
